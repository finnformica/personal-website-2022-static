Web Development: My Journey

After watching a podcast on how Oliver Cookson founded Myprotein through a simple static website I was inspired to learn more about web development. He spoke about how powerful and flexible of a tool a website is and in this age of digitalisation I wholeheartedly agree. Below are some takeaways from the web development projects I have embarked on.

My first website project
  • Visually unappealing.
  • Instrumental in teaching me how to develop a website.
  • Difficulty with using JavaScript.
  • Unable to make this simple snake game work.

My second website project
  • A simple shop with a cart to add items to.
  • Utilised local storage to retain cart items.
  • More Javascript was required which helped me become accustomed to JS.

My latest website project (current)

  • Uses Flask to serve the website.
  • Progress made from the first website.
  • Front-end skills require practice.
  • Difficulty passing variables from Flask to JavaScript and back again.
